# 경제성평가 연구 지원 센터 — 저장소 안내

이 저장소는 "경제성평가 연구 지원 센터"의 간단한 정적 웹 포털 템플릿을 제공합니다. 연구지원 서비스, 교육·워크숍 안내, 데이터·코드 공유, 컨설팅 신청 방법 등을 포함하도록 구성되어 있습니다.

## 포함된 파일
- `index.html` : 센터 소개 및 서비스, 예시 시각화가 포함된 메인 페이지
- `README.md` : 이 파일 — 설치, 커스터마이즈, 서비스 요청 가이드

## 센터가 제공하는 주요 서비스
1. 연구지원
   - Markov 및 개체 기반 모델 설계·구현
   - 비용-효과 분석(CEA), 비용-효용분석(CUA), 비용-편익분석(CBA)
2. 분석대행
   - 확률 민감도 분석(PSA), ICER 계산, 비용-효과 평면 및 CEAC 작성
3. 교육·워크숍
   - R/Python을 이용한 실습 중심 교육과정 제공
4. 데이터·코드 공유 및 검토
   - 재현 가능한 분석을 위한 리포지터리 구성, 코드 리뷰
5. 컨설팅 / 상담 예약
   - 프로젝트 단위 상담, 단기 협업

## 빠른 시작 (로컬)
1. 저장소를 클론하거나 파일을 다운로드:
```bash
git clone <REPO-URL>
cd <repo-folder>
```

2. 브라우저로 `index.html` 열기:
- 로컬 정적 서버 실행(옵션):
```bash
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
